# 🚀 BizEase – All-in-One Digital Growth Platform for Small Businesses

## 📌 Problem Statement

Small businesses struggle to establish, grow, and manage their digital presence due to:

* Lack of technical skills for website creation
* Inefficient marketing strategies within limited budgets
* Fragmented and underutilized SaaS tools

This leads to **lost opportunities, poor online visibility, and unnecessary recurring costs**.

---

## 💡 Solution

**BizEase** is an all-in-one SaaS platform designed to simplify digital growth for small businesses by providing:

* 🌐 No-code website creation
* 📢 Smart ad campaign optimization
* 💰 SaaS expense tracking & cost reduction
* 📊 Unified dashboard for complete business insights

👉 Everything is managed from a **single, easy-to-use platform**, eliminating the need for multiple tools.

---

## 🎯 Target Users

* Small business owners
* Local shop owners
* Startups with limited technical knowledge
* Entrepreneurs with low marketing budgets

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-username/bizease.git
cd bizease
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Frontend

```bash
npm run dev
```

### 4. Run Backend

```bash
cd backend
npm install
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in backend:

```
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
```
